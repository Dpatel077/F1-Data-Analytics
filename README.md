# 🏎️ Formula 1 Data Dashboard (2019-2025)

An interactive web dashboard built with Python and Streamlit to analyze Formula 1 race data across seven seasons. This project dives into historical race results to uncover insights about driver consistency, team reliability, track difficulty, and starting grid advantages.

## 📊 What This Project Analyzes

This dashboard uses Plotly to visualize the answers to 10 specific questions about modern F1 racing:

1. **The Grid Advantage:** How much does a driver's starting position actually predict their final finish?
2. **Team Reliability:** Which constructors threw away the most points due to DNFs and mechanical failures?
3. **Fastest Lap Strategy:** Do race winners usually take the bonus point, or do midfield drivers steal it at the end?
4. **Track Overtaking Difficulty:** Which circuits are historically the hardest to pass on?
5. **Biggest Movers:** Which drivers consistently overachieve and make up places on Sunday?
6. **Teammate Head-to-Heads:** Who is dominating the internal team battles in points and average finish?
7. **The Midfield Battle:** How tight is the points spread for the midfield teams compared to the top 3?
8. **Consistency vs. Peak Performance:** Does steady point-scoring beat a volatile season of wins and DNFs?
9. **Survival Rate:** Does just surviving the race and finishing laps correlate heavily with championship success?
10. **Podium Diversity:** Did a few teams completely hog the podiums, or was the hardware shared across the grid?

## 🛠️ Tech Stack

* **Python:** Core data processing
* **Pandas:** Data wrangling, cleaning, and aggregation
* **Streamlit:** Web framework and interactive UI widgets
* **Plotly Express:** Interactive charting and data visualization
* **Statsmodels:** Statistical trendlines (Ordinary Least Squares)
